# 权限说明

- 系统权限角色共两个：项目管理员和项目成员可进入作业开发和作业运维模块。非项目成员只能查看项目信息。项目管理员可对项目进行编辑，和成员管理，默认项目创建人即项目管理员。
- 当登录用户没有项目时，可以根据实际情况选择申请项目权限，若平台没有对应的项目可以联系管理员添加项目。
